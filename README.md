## Simon Santa Claus
Application that lets you and your friends organize a Christmas lottery. It offers really cool chat and communication features.

# Technologies
- TypeScript
- React.js
- Next.js
- Supabase - PostgreSQL
- Prisma
- CSS/SCSS

- next-auth
- axios
- bcrypt
- dayjs
- dayjs-business-days2
